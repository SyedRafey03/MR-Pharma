# MR PHARMA - Pharmaceutical Distributors

Welcome to the official website repository for **MR PHARMA**, a trusted pharmaceutical distributor.

## 🚀 What kind of website is this?

This is a **corporate informational website (static landing page)**. Its primary purpose is to act as a digital business card and company profile. It provides visitors with essential information about the company, such as who they are, the services they offer, their legal credentials, and how to get in touch. 

## 🚀 Overview

This repository contains the source code specifically designed to showcase the MR PHARMA brand. It serves to inform healthcare providers, pharmacies, and clinics about the distribution network's capabilities and acts as a central hub for their online presence.

## 🛠️ Built With

- HTML5 / CSS3
- [Bootstrap](https://getbootstrap.com/)
- AOS (Animate On Scroll)
- GLightbox
- Swiper

## 📂 Project Structure

```text
MR-Pharma-master/
├── assets/          # Contains CSS, JS, Images, and Vendor libraries
├── index.html       # Main landing page
└── README.md        # Project documentation
```

## 🚀 Getting Started

To view the website locally:

1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. No build tools or package managers are required as this is a static HTML site.
